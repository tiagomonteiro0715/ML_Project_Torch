# Project abstract

RNNs and their variants, such as GRUs and LSTMs, have traditionally been the go-to models for language-related tasks and time series analysis. In 2017, the introduction of the transformer neural network architecture addressed the long-standing limitations of strictly sequential data processing and incomplete attention mechanisms. However, due to transformers’ quadratic time complexity, alternative models like Mamba and newer variants such as xLSTM have outperformed them on some tasks in the Long Range Arena benchmark. All RNNs operate under the assumption that their hidden state behaves uniformly with respect to the tokens being processed. Put differently, the weights associated with the hidden units in an RNN are updated at the same rate as the sequence of input data arrives. A Google study described machine learning as a field of sequential, parallel, and nested optimization processes, and suggested that a continuous memory spectrum can improve a model’s memory. This project investigates whether the continuous memory spectrum in RNNs, combined with It and gate mixes and other architectures (such as CMS-augmented Mamba, vision models, and residual connections that strengthen backpropagation), can improve performance. At present, a transformer equipped with residual blocks, a forget gate, and linear attention is matching the performance of linear transformers on several benchmarks. We are now assessing how CMS can be applied within LSTM models.
